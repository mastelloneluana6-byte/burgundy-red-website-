# LUXEYE - Premium Skincare Landing Page

A high-end, responsive landing page for a fictional luxury skincare brand called **LUXEYE**, featuring their flagship "Peptide Collagen Eye Cream". This project was built to demonstrate modern web design principles, fluid animations, and high-converting UI/UX patterns.

## 🌟 Features

- **Premium Aesthetics**: Dark modes, elegant typography (Inter), and carefully selected color palettes (Burgundy & Deep Black).
- **GSAP Animations**: Fluid scroll-triggered animations (fade-ups, slide-ins, scale-ins) that bring the page to life as the user scrolls.
- **Dynamic Preloader**: A sleek loading screen with a pulsing burgundy glow effect before the page content is revealed.
- **Interactive FAQ Accordion**: A smooth, animated accordion section for Frequently Asked Questions.
- **Sticky Mobile Buy Bar**: A sticky call-to-action bar that appears only on mobile devices to maximize conversion rates.
- **Interactive Support Chatbot**: A mocked, interactive floating chatbot widget that greets users and responds to messages.
- **Fully Responsive**: Optimized for desktop, tablet, and mobile viewing.

## 🛠️ Technologies Used

- **HTML5**: Semantic markup.
- **Vanilla CSS3**: Custom properties, Flexbox, Grid, and Media Queries (No external CSS frameworks used).
- **Vanilla JavaScript (ES6)**: DOM manipulation, event listeners, and chatbot logic.
- **GSAP & ScrollTrigger**: For high-performance, scroll-linked animations.
- **FontAwesome**: For scalable vector icons.
- **Google Fonts**: Inter font family.

## 🚀 How to Run Locally

Since this is a static frontend project, no complex build tools or servers are required.

1. Clone this repository:
   ```bash
   git clone https://github.com/your-username/luxeye-landing-page.git
   ```
2. Navigate to the project directory:
   ```bash
   cd luxeye-landing-page
   ```
3. Open `index.html` directly in your favorite web browser.
   - Alternatively, you can use a local server like VS Code's "Live Server" extension for a better development experience.
